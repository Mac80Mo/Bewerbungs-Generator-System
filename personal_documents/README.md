# Persönliche Dokumente

Legen Sie hier Ihre persönlichen Dokumente ab, die für die automatische Extraktion verwendet werden sollen:

## Unterstützte Dateiformate
- PDF (`.pdf`)
- Word-Dokumente (`.docx`, `.doc`)
- Textdateien (`.txt`)
- Markdown (`.md`)

## Beispiele für nützliche Dokumente
- Lebenslauf / CV
- Arbeitszeugnisse
- Ausbildungszeugnisse / Abschlusszeugnisse
- Zertifikate
- Fortbildungsnachweise
- Bewerbungsschreiben (als Referenz)

## Verwendung
1. Dokumente in diesen Ordner kopieren
2. `python extract_personal_data.py` ausführen
3. Modell aus der Liste der verfügbaren Ollama-Modelle auswählen
4. Extraktion läuft automatisch

## Datenschutz
Alle Daten bleiben lokal auf Ihrem Rechner. Es werden keine Daten in die Cloud gesendet.
